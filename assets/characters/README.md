# 外部キャラクター画像の追加方法

このフォルダに画像ファイル（JPEG / PNG / WebP）を置き、
`manifest.json` に項目を追加すると、ゲームのキャラクター選択画面に自動で表示されます。

## 手順

### 1. 画像ファイルを置く
このフォルダ (`assets/characters/`) に画像をコピー。

**推奨サイズ**: 256×256px 以上の正方形に近いもの。背景が透過（PNG）だとキレイに表示されます。

### 2. manifest.json にエントリを追加

```json
{
  "id": "img_myhero",                     // 一意のID（他と被らないように）
  "name": "マイヒーロー",                  // 表示名
  "subtitle": "オリジナル・バランス型",      // サブタイトル
  "image": "myhero.png",                  // このフォルダ内のファイル名
  "baseSpeed": 6,                          // 初期速度 (4〜8推奨)
  "growth": 0.3,                           // レベルごとの加速 (0.2〜0.4推奨)
  "speedCap": 13,                          // 速度上限 (10〜15推奨)
  "themeColor": "#deb060"                 // UIカラー (HEX)
}
```

### 3. ブラウザをリロード
キャラ選択画面に新しいキャラが出ます！

---

## ゲーム画面から追加する方法（手軽）

キャラ選択画面の右下にある「**＋ 画像でキャラ追加**」ボタンをタップ → 画像ファイルを選択。
端末の localStorage に保存され、次回以降も選択できます。
※ ただしブラウザのキャッシュをクリアすると消えます。永続化したい場合は上記の手順でアセット化してください。

---

## ファイル命名のコツ
- 英数字 + ハイフン推奨 (`mychar.png`, `hero-v2.jpeg`)
- スペース・日本語はURLエンコードされるので避けた方が無難
